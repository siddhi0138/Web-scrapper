# Static vs Dynamic DOM

## Overview
Understanding the difference between static and dynamically rendered content.

## Static DOM
- HTML served directly from server
- No JavaScript execution required
- Content visible in page source
- Easier to scrape

## Dynamic DOM
- Content generated by JavaScript
- DOM rendered after page load
- Content not in page source
- Requires browser rendering

## Detection Methods
- Inspect page source
- Monitor network requests
- Check for JavaScript frameworks
- Analyze load patterns

## Scraping Implications
- Static: Use HTTP libraries
- Dynamic: Use headless browsers
