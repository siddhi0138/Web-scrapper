# HTTP Keywords & Concepts

Key HTTP concepts relevant to web data extraction:

- GET / POST: Request methods
- Headers: User-Agent, Accept, Authorization
- Status Codes:
  - 200 OK
  - 301/302 Redirects
  - 403 Forbidden
  - 429 Too Many Requests
  - 5xx Server Errors

Understanding these is critical for diagnosing scraping failures.
