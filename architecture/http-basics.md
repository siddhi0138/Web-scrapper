# HTTP Basics

## Overview
Fundamentals of the HTTP protocol for web data extraction.

## Key Concepts
- HTTP methods (GET, POST, PUT, DELETE)
- Status codes
- Headers and metadata
- Request/Response lifecycle

## HTTP Methods
- **GET**: Retrieve data
- **POST**: Submit data
- **PUT**: Update data
- **DELETE**: Remove data

## Common Status Codes
- 200: OK
- 404: Not Found
- 403: Forbidden
- 500: Server Error
